## Installation

<Tabs>
	<TabItem label="bun">```bash bun add -D eslint-plugin-cease-nonsense ```</TabItem>
	<TabItem label="npm">```bash npm install -D eslint-plugin-cease-nonsense ```</TabItem>
	<TabItem label="pnpm">```bash pnpm add -D eslint-plugin-cease-nonsense ```</TabItem>
</Tabs>

## Configuration

<Steps>

1. **Add to your ESLint config**

    ```typescript
    import ceaseNonsense from "eslint-plugin-cease-nonsense";

    export default [
    	ceaseNonsense.configs.recommended,
    	// ... your other configs
    ];
    ```

2. **Run ESLint**

    ```bash
    bun run eslint .
    ```

</Steps>

## What's Included

The `recommended` config enables all rules with their default severity levels. Most rules are set to `warn`, with a
subset set to `error` for critical issues.

See the [Configuration Reference](/eslint-cease-nonsense-rules/configuration/) for details on available configs and
per-rule severity settings.
