Prefer EnumItem values over magic strings/numbers.

## Rule Details :badge[Roblox]{variant=note}

In `roblox-ts`, you can often pass an enum value as its string name or numeric value. It works at runtime, but you lose
type safety and IDE features like "Go to Definition" and "Find References". This rule prefers the explicit form
(`Enum.ScaleType.Slice`) over magic strings (`"Slice"`) or numbers (`1`).

<Aside type="tip" title="Self-Documenting Code">
	`Enum.UIFlexMode.Fill` is clearer than `"Fill"`.
</Aside>

## Refactoring Steps

<Steps>
1. **Identify Magic Literals**
   Locate strings or numbers being used in places where a Roblox Enum is expected (e.g., JSX properties like `ScaleType`).

2. **Search Enum Global** Look up the corresponding Enum name in the
   [Roblox Enum Browser](https://create.roblox.com/docs/reference/engine/enums).

3. **Update Reference** Replace the literal with the full `Enum.Category.Member` path.

4. **Auto-Fix** Most violations can be fixed automatically by running `eslint --fix`.

{/* oxfmt-ignore */}

</Steps>

## Examples

<Tabs>
	<TabItem label="❌ Incorrect" icon="close">
		### Magic Values
		Bans patterns that use strings or numbers where enums are expected.

    	<Code
    		code={[
    			"// String literal",
    			'<uiflexitem FlexMode="Fill" />',
    			"",
    			"// Number literal",
    			"<Halftones nativeProperties={{ ScaleType: 1 }} />",
    			"",
    			"// In function arguments",
    			'setFlexMode("Fill");',
    		].join("\n")}
    		lang="tsx"
    	/>
    </TabItem>
    <TabItem label="✅ Correct" icon="check">
    	### Explicit Enums
    	Use the built-in `Enum` global for all property assignments and arguments.

    	<Code
    		code={[
    			"// Explicit enum usage",
    			"<uiflexitem FlexMode={Enum.UIFlexMode.Fill} />",
    			"",
    			"// Explicit enum in objects",
    			"<Halftones nativeProperties={{ ScaleType: Enum.ScaleType.Slice }} />",
    			"",
    			"// In function arguments",
    			"setFlexMode(Enum.UIFlexMode.Fill);",
    		].join("\n")}
    		lang="tsx"
    	/>
    </TabItem>

</Tabs>

## Configuration

<CardGrid stagger>
	<Card title="fixNumericToValue" icon="setting">
