## Quick Start

<Steps>

1. **Install the package**

    <Code code="bun add -D eslint-plugin-cease-nonsense" lang="bash" title="Terminal" />

2. **Add to your ESLint config**

    <Code
    	code={[
    		"import ceaseNonsense from 'eslint-plugin-cease-nonsense';",
    		"",
    		"export default [",
    		"\tceaseNonsense.configs.recommended,",
    		"\t// ... your other configs",
    		"];",
    	].join("\n")}
    	lang="typescript"
    	title="eslint.config.ts"
    />

3. **Run ESLint**

    <Code code="bun run eslint ." lang="bash" title="Terminal" />

</Steps>
